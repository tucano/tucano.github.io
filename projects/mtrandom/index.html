MTRandom

An alternative Random library for Unity3d designed to generate uniform Pseudo-Random deviates
April 17, 2012 · Davide Rambaldi

MTRandom#

In Unity3d there is already a Random number generator based on the platform-specific random generator. Here we present an alternative Random library for Unity3d designed to generate uniform Pseudo-Random deviates. The library use a fast PRNG (Mersenne-Twister) to generate: Floating Numbers in range [0-1] and in range [n-m], Vector2 and Vector3 data types. The library comes with some special functions designed specifically for a game design framework: Shuffle Bag, Dice, Random Color.
